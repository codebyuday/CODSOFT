# Python Applications Collection

A collection of command-line applications built with Python.

## Applications

### 1. To-Do List Manager

A task management application to help you organize your daily tasks.

**Features:**
- Add tasks with priority levels (high/medium/low)
- View all tasks in a formatted table
- Update task title and priority
- Delete tasks
- Mark tasks as complete
- View statistics (total, completed, pending, high priority)

**Run:**
```bash
cd todo_app
python todo_app.py
```

### 2. Contact Book

A contact management application to store and manage your contacts.

**Features:**
- Add contacts (name, phone, email, address)
- View all contacts in table format
- Search contacts by name or phone number
- Update contact details
- Delete contacts with confirmation
- View contact statistics

**Run:**
```bash
cd contact_book
python contact_book.py
```

### 3. Rock-Paper-Scissors Game

A classic Rock-Paper-Scissors game against the computer.

**Features:**
- Choose rock, paper, or scissors
- Random computer selection
- Win/lose/tie detection
- Live scoreboard tracking
- Play multiple rounds

**Run:**
```bash
cd rps_game
python rps_game.py
```

### 4. Password Generator

A secure password generator with customization options.

**Features:**
- Custom password length and character types
- Quick password presets (Weak/Moderate/Strong/Very Strong)
- Batch generate multiple passwords
- Password strength analysis
- Password history tracking

**Run:**
```bash
cd password_generator
python password_generator.py
```

## Requirements

- Python 3.x
